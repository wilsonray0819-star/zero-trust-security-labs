# Zero Trust Security Labs

This repository documents a series of hands-on IT and cybersecurity labs focused on implementing Zero Trust principles using Microsoft and cloud-based technologies.

## Overview
These labs demonstrate practical experience with:
- Identity and Conditional Access
- Endpoint security and device compliance
- Network segmentation and Zero Trust Network Access (ZTNA)
- Application authentication and access controls
- Data classification and data loss prevention (DLP)
- End-to-end Zero Trust architecture design

## Tools and Technologies
- Microsoft Entra ID (Azure AD)
- Conditional Access
- Microsoft Intune
- Microsoft Defender
- Azure networking components
- Microsoft Purview
- Zero Trust architecture principles

## Lab Structure
Each lab folder contains:
- A README explaining objectives and configuration
- Screenshots demonstrating implementation
- Diagrams where applicable

## Purpose
This repository serves as a public portfolio to demonstrate applied IT and cybersecurity skills for entry-level IT, security, and cloud roles.
